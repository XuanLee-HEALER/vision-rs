# 开发命令

## 基本命令

### 开发服务器

```bash
npm run dev
```

启动开发服务器，默认端口 3000

### 构建

```bash
npm run build
```

构建生产版本

### 启动生产服务器

```bash
npm run start
```

启动生产服务器

### 代码检查

```bash
npm run lint
```

运行 ESLint 检查

## Git 命令

```bash
git status           # 查看状态
git add .            # 添加所有更改
git commit -m "msg"  # 提交
git push             # 推送
```

## 系统命令 (macOS/Darwin)

```bash
ls -la              # 列出文件
cd <dir>            # 切换目录
grep -r "pattern" . # 搜索文件内容
find . -name "*.ts" # 查找文件
```
