# 任务完成检查清单

## 完成任务前必须执行

### 1. 代码检查
```bash
npm run lint
```
确保没有 ESLint 错误

### 2. 构建测试
```bash
npm run build
```
确保项目能成功构建

### 3. 本地测试
```bash
npm run dev
```
在浏览器中检查功能是否正常

## 代码审查要点
- [ ] TypeScript 类型正确
- [ ] 遵循 Tailwind CSS 类名约定
- [ ] 遵循 Catppuccin Macchiato 配色
- [ ] 组件结构清晰
- [ ] 无控制台错误

## 提交前
- [ ] 运行 `npm run lint` 无错误
- [ ] 运行 `npm run build` 成功
- [ ] 页面在本地正常显示
