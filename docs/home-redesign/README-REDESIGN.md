## 💡 设计理念

### 杂志式布局的核心

1. **打破常规** - 非对称网格，拒绝中规中矩
2. **大胆留白** - 间距是设计的一部分，不是浪费
3. **视觉流动** - 引导用户视线，创造阅读节奏
4. **克制使用装饰** - 每个元素都有目的，不为装饰而装饰

### 极简主义原则

- 移除所有不必要的图标和装饰（包括 Emoji）
- 用排版和留白创造视觉冲击
- 让内容本身成为设计
